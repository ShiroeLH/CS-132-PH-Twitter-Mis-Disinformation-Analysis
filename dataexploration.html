PREPROCESSING

Some steps and techniques were used to transform the gatehred raw data into a suitable format for analysis and data modeling.

Handling missing values/ensuring no missing values

In this step, any missing values in the dataset are identified and handled appropriately. The columns "Screenshot," "Location," "Views," and "Remarks" are dropped from the dataset as they contain missing values. For accounts without a bio, an empty string ("") is used to represent the missing value. The columns that has null values are dropped to create df_clean.

Ensuring formatting consistency

The date formatting inconsistencies are addressed to ensure uniformity across the dataset. Pandas default datetime format is used: YYYY-MM-DD HH:MM:SS. This is saved as .Date posted feature in df_clean.

Categorical data encoding

To facilitate data exploration, the categorical feature "Account type" was encoded into numerical values: 0 for "Anonymous," 1 for "Identified," and 2 for "Media." This transformed value is saved as Account type feature in df_clean. Additionally, the dataset's emoji data was transformed into aliases using the emoji library.

Tokenization and lower casing

In order to analyze the tweets effectively, the WordCloud technique is employed to break down the tweets into individual words or tokens, which are then examined separately. To identify the colleges and universities that are most targeted in the red tag-related tweets, a dictionary is constructed. This dictionary contains lowercase acronyms or abbreviations of various educational institutions, with their corresponding full names as the values. The frequency of each college or university is determined by counting how many times its lowercase representation appears among the tokenized tweets. By analyzing the frequency of these institutions in the tokens, we can identify the ones that are mentioned or targeted most frequently in the red tag-related tweets.

To enhance analysis and modeling efficiency, the removal of stop words is employed. The WordCloud library offers a predefined set of English stop words such as "and," "the," and "in," which are commonly used but do not contribute significant contextual information. These words are excluded from the dataset to prioritize more relevant terms and reduce noise. However, the word "up" is manually excluded from the English stop words list to avoid erroneously removing instances referring to the University of the Philippines. Additionally, Filipino stop words are considered, sourced from the stopwords-tl repository on GitHub. By eliminating both English and Filipino stop words, the analysis focuses on more meaningful and informative terms found in red tag-related tweets regarding colleges and universities.

Interpolation

To aggregate the data, the date and time is resampled to a monthly frequency and then the mean value is calculated for each month. To handle any missing values within a time period, the interpolation method using interpolate() with method='spline' is used to perform spline interpolation on the data. Spline interpolation is a mathematical technique used to fit a smooth curve (spline) through the data points through piecewise-defined function that consists of multiple polynomial segments.

To simplify and organize continuous date and time data, it was divided into Monthly, Yearly intervals or bins by using groupby(). A specific use of this collecting the number of tweets per month using ...groupby(pd.Grouper(key="Date posted", freq="1M")).count(), and collecting number and mean of engagements per year ...groupby(pd.Grouper(key="Date posted", freq="1Y")).sum() and ...groupby(pd.Grouper(key="Date posted", freq="1Y")).mean()

VISUALIZATION

Various visualization techniques are employed to gain insights from the data and communicate findings effectively. Some common visualization methods include:

Scatterplots/histograms

These plots show the distribution and relationship between two continuous variables. Scatterplots display the relationship between variables, while histograms illustrate the distribution of a single variable.:

The scatterplot above displays the relationship between the number of retweets and the length of the tweet. Tweets with retweet count >= 50 would most likely fall under a tweet length range of 200-300. We should take note, however, that most tweets, regardless of tweet length, hold a retweet count of 0.

The scatterplot above displays the relationship between the content type and the number of engagements of the tweet. The content with a large number of engagements (approx > 1000) tend to fall under the Rational content type despite the top content type being Emotional with 113 tweets possessing this type, followed by 59 and 1 of Rational and Transactional, respectively. This can infer that most twitter audiences tend to react more on a tweet that is informational, functional, educational, or current event. Tweets that trigger rational thinking may engage more audiences than those that trigger emotions or invoke transactions. Note that the number of engagements of a tweet is defined in this study as the sum of Likes, Replies, Retweets, and Replies a tweet has obtained.

Heat maps

Heat maps provide a visual representation of the correlation between different features. This is particularly useful for identifying patterns or relationships between variables in a tabular format.

The matrix above is a visual representation of the correlation between number of Likes, Replies, and Retweets, and Length of tweet. Pearson coefficient values show a negligible correlation (r < 0.10) among these variables. This shows that a lengthy tweet does not necessarily imply a high number of Likes, a high number of Replies, and a high number of Retweet, and so forth.

The matrix above is a visual representation of the correlation between the tweet's Engagements and the number of Followers of the tweet author. Pearson coefficient values show a weak correlation (0.10 < r < 0.39) among these variables. This implies that accounts with a greater number of followers may, but not strictly, tend to have more engagements than those accounts with less followers..

Bar/swarm/violin plots

These plots are used to compare categorical variables or visualize the distribution of a continuous variable across different categories. Bar plots display the frequency or count of each category, while swarm plots show individual data points along with the distribution. Violin plots combine a box plot and a kernel density plot to display the distribution of data.

This bar graph shows the frequency of tweets at a tweet length interval of 10 characters. Most number of red-tagging tweets are of length within the range of 270-279 characters. Shorter tweet length may not be sufficient to red-tag university students.

This Word Cloud serves as a visual representation of the frequency of Emojis within the gathered Account bio. Based on the visualization, some of the most common emojis are evil_monkey, white_heart, red_heart, green_heart, oncoming_fist and victory_hand.

FEATURE ANALYSIS

Features within a dataset were examined to gain insights and make informed decisions about their relevance and usefulness for the topic.

Feature generation

School and Tweet Length features based on existing data were added to help in further analysis. The "School" (df_clean['school']) feature was generated by extracting values from the tweets using a manually created dictionary of college acronyms and related words. The "Tweet Length" (df_clean['Length']) feature represents the length of each tweet, providing information about the number of characters or words used in the tweet.
